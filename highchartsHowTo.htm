Highcharts uses either jQuery, MooTools or Prototype for some common JavaScript tasks. You need to include the JavaScript files in the section of your web page. We use jQuery in our project, so this tutorial will cover jQuery specific details only. Use this code to include Highcharts with jQuery:

<script src="jquery/js/jquery-1.6.4.min.js"></script>
<script src="highcharts/js/highcharts.js"></script>

First Chart

Include the above two files to your html page. Then include the following script to that page.

var chart1;
$(document).ready(function() {
 chart1 = new Highcharts.Chart({
  chart: {
   renderTo: 'container',
   type: 'bar'
  },
   title: {
   text: 'Fruit Consumption'
  },
  xAxis: {
   categories: ['Apples', 'Bananas', 'Oranges']
  },
  yAxis: {
   title: {
    text: 'Fruit eaten'
   }
  },
  series: [{
   name: 'Ramasamy',
   data: [1, 0, 4]
  }, {
   name: 'Kuppusamy',
   data: [5, 7, 3]
  }]
 });
});

Add a div in your webpage. Give it an id refering to the renderTo option in the above script, and set a specific width and height which will be the width and height of your chart.

<div id="container" style="width: 100%; height: 400px"> </div>
